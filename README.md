## Installation

1. Clone the project:
```bash
git clone https://github.com/matansegev/NoteMe.git
cd NoteMe
```

2. Install dependencies:
```bash
npm install
```

3. Run development server:
```bash
npm run dev
```

4. Open browser at:
```
http://localhost:5173
```
